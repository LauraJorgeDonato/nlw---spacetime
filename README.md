## 💻 Projeto

Esse é um projeto Web Responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias

Foi desenvolvido durante o NLW do RocketSeat com as seguintes tecnologias:

- HTML
- CSS
- Git e GitHub
